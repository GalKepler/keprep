============================================================================
KePrep: PreProcessing Strauss Neuroplasticity Brain Bank dMRI data
============================================================================

KePrep is a Nipype_-based pipeline for the preprocessing of dMRI data associated with the Neuroplasticity Brain Bank (SNBB).
